  1. /EthCtrlConfig/EthCtrlEnableMii：This parameter defines if the Media Independent Interface (MII) for transceiver access is enabled.Restrictions: Must be enabled for transceiver usage.如果我们工程中有使用transceiver的话，则这个参数一定要使能。
    关于MII的介绍请参考文章【Eth 02 -MAC如何访问PHY】。

  2. EthCtrlConfig/EthCtrlEnablePtp：This parameter defines if Precision-Time-Protocol support is enabled or not. PTP is mostly used in combination with Audio Video Bridging (AVB)。如果使用时间同步功能则使能这个选项。它还要依赖于下面这个参数EthPtpRefClock。

  3. EthCtrlConfig/EthPtpRefClock：This value specifies the frequency of the PTP reference clock.定义PTP引用的时钟，一般定义为晶振时钟源相关的时钟源。
