1 配置AutoSAR通信模块

预热知识点：Davinci工具中的通信模块一般包含Domain-specific配置编辑器（configuration editors），用于配置通信相关模块、ECU 总线控制器、协议数据单元、数据信号和传输协议的一般参数。
注意：Davinci工具中的配置器包括一般编辑器（Basic editors）和Domain-specific配置编辑器（Domain-specific configuration editors）。

  • 特定功能模块的配置编辑器，它提供了有关 ECUC 的优化视图。

  • 基本编辑器，提供有关 ECUC 的基本视图。

    下面模块的配置都是基于Domain-specific configuration editors。

1.1 配置Communication General

定义基于通信的模块的核心功能设置。以Can为例，按照如下步骤配置：

  1. 打开 CAN | Miscellaneous选项标签。
  2. 设置模块引用的计数/计时源，通过设置Counter Ref参数来实现，点击[…]打开窗口后选择系统定时器SystemTimer。
  3. 将CAN中断类型设置为第二类中断CATEGORY 2，即设置 Interrupt Category 为 CATEGORY 2。
  4. 配置中断的锁定级别设定为驱动级别,即设置Interrupt Lock为DRIVER。

1.2 配置总线控制器

  1. 打开CAN Controllers | 。
  2. 定义控制器默认波特率Controller Default Baudrate。
  3. 定义CPU 时钟参考CPU Clock Ref。

    如果如果没有可用的 CPU Clock Ref，请在Select Reference Target视图中添加一个新的参考。如下图所示，按照1-2-3的步骤来选择特定的参考。
